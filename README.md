## Описание
GUI-приложение для хранения информации о фильмах с возможностью добавления, редактирования, удаления и фильтрации. Данные сохраняются в JSON файл.

## Функциональность
- ✅ Добавление фильмов (название, жанр, год, рейтинг)
- ✅ Отображение списка в таблице
- ✅ Фильтрация по жанру и году
- ✅ Удаление и редактирование записей
- ✅ Сохранение/загрузка из JSON
- ✅ Валидация ввода (год - число, рейтинг 0-10)

## Технологии
- Python 3.x
- Tkinter (GUI)
- JSON (хранение данных)

# Запуск приложения
python movie_library.py
